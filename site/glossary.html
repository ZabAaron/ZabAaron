Glossary

The Heading Element
The heading element, represented by <h>, followed by a number inside the closing bracket, is used to define headings of different levels, from h1 to h6. This is not the same as font size and is used to define the thickness of the text characters.
The Nav Element
The nav element, represented by <nav>, is used to denote navigational content. This includes elements such as "Home," "Links," and "Glossary" links.
The Blockquote Element
The blockquote element, represented by <blockquote>, is used for quoting a larger section of text from another source.
The Paragraph Element
The paragraph element, represented by <p>, is used to define blocks of text content. It helps in organizing content into seperate sections.
The List Element
The list elements include <ul> (unordered list) and <ol> (ordered list), which contain list items (<li>) which can be used to structure content in bullet points or numbered sequences.
